    Opportunities

    Explore curated opportunities designed to support underrepresented students in tech — including internships, mentorships, scholarships, and events.

    AfroTech Conference

    Type: Event

    Description: AfroTech is a premier tech conference that brings together Black tech professionals, entrepreneurs, and students to network, learn, and explore career opportunities in the tech industry.

    Women Who Code

    Type: Event, Mentorship

    Description: Women Who Code hosts global events that support women in tech through networking, technical workshops, and career development to help close the gender gap in the industry.

    Black Girls Code

    Type: Program

    Description: Black Girls Code empowers young Black girls to become innovators in STEM by providing hands-on workshops, coding programs, and events that foster skills and confidence in tech.

    Tech Bridge

    Type: Program

    Description: TechBridge is a nonprofit that connects underserved communities with technology solutions, offering workforce development programs and tech-focused training to bridge the digital divide.

    Break Through Tech

    Type: Program, Internship

    Description: Break Through Tech works to increase gender diversity in tech by offering programs like Sprinternships™, career development workshops, and networking opportunities for women and nonbinary students in computing.

    Rewriting the Code

    Type: Program

    Description: Rewriting The Code is a nonprofit organization that connects collegiate women with career opportunities in tech through mentorship programs, job placements, and a network of professional women in the industry.

    Asian Women Giving Circle

    Type: Program

    Description: The Asian Women Giving Circle supports Asian American women-led organizations by providing funding, leadership, and resources to initiatives that address social justice issues, with a focus on community empowerment and advocacy.

    Out In Tech

    Type: Program

    Description: Out in Tech is a nonprofit organization that connects LGBTQ+ people in the tech industry through networking events, mentorship opportunities, and career development programs, aiming to support the community’s growth in tech fields.

    Techqueria

    Type: Program

    Description: Techqueria is a nonprofit organization that provides a community and resources for Latinx professionals in the tech industry, offering networking events, career development, and support for those pursuing tech careers.

    About

    DiversiTech was created as part of an advocacy-focused technology project to address the ongoing underrepresentation of marginalized groups in the tech industry. Developed by Shaina Silawan, a software engineer committed to equity in tech, this site serves as a centralized hub for discovering life-changing opportunities. By making internships, scholarships, mentorships, and events more accessible, DiversiTech helps bridge the gap in access and representation.

    The platform is currently in beta, with ongoing development to improve how opportunities are gathered and displayed. Future implementations will focus on automating listings, enhancing search and filtering, implementing back end, and creating a more personalized user experience to better serve the community.
